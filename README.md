涂鸦鸿蒙版智能生活 App SDK 适用于5.1及以上手机系统。

## 功能说明
SDK 主要包括以下功能：

- 用户账号：支持手机号码、邮箱的注册和登录、账号密码重置、登录 Session 失效处理等全方位用户账号相关的功能。详情请参考 用户账号。

- 家庭管理：包含家庭和房间的管理、家庭共享等功能。详情请参考 家庭管理。

- 设备管控：包含了设备配网、设备控制、设备的状态上报、群组、定时任务、固件升级等功能。


## 垂直品类 SDK
智能生活 App SDK 提供一些垂直品类的 SDK，包括 智能摄像机 SDK、智能门锁 SDK、扫地机 SDK、智慧出行 SDK 等。它们基于智能生活 App SDK，主要针对垂直品类特有的功能进行封装，方便垂直品类产品功能的快速开发。



## SDK 示例
涂鸦提供了开源的鸿蒙Sample。您可以基于 Sample 快速构建 Smart App，或者通过添加断点等方式，解决开发过程中遇到的问题。
